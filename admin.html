Publicar as alterações

As alterações feitas acima ficam apenas neste navegador até serem publicadas. Descarregue o novo ficheiro data.js e substitua o ficheiro js/data.js no seu repositório do GitHub (depois faça commit + push — ou arraste o ficheiro directamente na página do GitHub).
